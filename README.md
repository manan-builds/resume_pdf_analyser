Resume PDF Analyzer & Skill Extractor
An AI-powered Resume Analyzer that extracts skills from PDF resumes, predicts suitable job roles, and stores candidate data in a MySQL database using a simple Gradio web interface.

Features

Upload Resume PDF  
Extract resume text  
Detect skills using keyword matching  
Predict role (ML Engineer / Backend Developer / Data Scientist)  
Generate suitability score  
Store candidate profile in MySQL  
Simple Gradio UI  

Tech Stack:

Python
Gradio
Scikit-learn
PDFPlumber
MySQL
Joblib
